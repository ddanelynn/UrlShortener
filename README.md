# UrlShortener

### Features
* Shorten a given URL
* Easy copy to clipboard
* Raise invalid URL warning
* Responsive UI

### Tech Stack
ReactJs - Express - NodeJs - MySql

### Setup Instruction
* Navigate to client directory, run `npm start`
* Navigate to server directory, run `nodemon index.js`
* You must have mySql installed
